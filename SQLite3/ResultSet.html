class SQLite3::ResultSet

Overview

The ResultSet object encapsulates the enumerability of a queryâ€™s output. It is a simple cursor over the data that the query returns.

Typical usage is:

require "sqlite3"

db = SQLite3::Database.new("foo.db")
stmt = db.prepare("select * from person")
result_set = stmt.execute
while result_set.next
  p result_set.to_a
end
stmt.close
db.close
